# Security Policy

## Supported Versions

We aim to support the latest released version of `main`. Please always
test against the most recent commit before reporting issues.

## Reporting a Vulnerability

If you believe you have found a security vulnerability in this project, please
**do not** open a public issue. Instead, report it privately via GitHub's
[Private Vulnerability Reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability)
or email the maintainers directly.

We will acknowledge your report within 72 hours and aim to provide a fix or
mitigation as quickly as possible.

Please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any proof-of-concept code or screenshots
- Your suggested remediation, if any

Thank you for helping keep this project and its users safe.
